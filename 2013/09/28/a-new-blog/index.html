A New Blog

So you found my new blog eh ! Cool.

Blogging through services like wordpress got kinda boring so was searching for alternatives when I found Hexo. Awesome stuff !! Very flexible and easy to use. Hosting through GitHub pages. Supports GitHub flavoured markdown and fully compatible with Jekyll/Octopress. Heck you can even install plugins for Jade. Sounded perfect so decided to build my new blog on it. Do check it out.
